Our Coding Topics

HTML

HTML stands for HyperText Markup Language. It is used to create the structure and content of a website, including headings, text, images and links.

CSS

CSS stands for Cascading Style Sheets. It is used to style websites by changing colors, fonts, spacing and layouts.

JavaScript

JavaScript makes websites interactive. It can be used to create buttons, calculations, quizzes and many other functions.

API

An API allows different programs and websites to exchange information and communicate with each other.
