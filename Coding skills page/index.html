Unsere Coding-Themen

HTML

HTML bedeutet HyperText Markup Language. Mit HTML erstellt man den Aufbau und den Inhalt einer Webseite, zum Beispiel Überschriften, Texte, Bilder und Links.

CSS

CSS bedeutet Cascading Style Sheets. Mit CSS kann man das Aussehen einer Webseite verändern, zum Beispiel Farben, Schriftgrößen und Abstände.

JavaScript

JavaScript macht eine Webseite interaktiv. Damit kann man zum Beispiel Buttons, Berechnungen, Quizze und andere Funktionen programmieren.

API

Eine API ermöglicht den Austausch von Daten zwischen verschiedenen Programmen oder Webseiten.
