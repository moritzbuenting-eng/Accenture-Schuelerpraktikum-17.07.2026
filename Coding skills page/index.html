Our Coding Topics

HTML

HTML stands for HyperText Markup Language. It is used to create the structure of a website.

CSS

CSS stands for Cascading Style Sheets. It is used to design the layout, colors and fonts of a website.

JavaScript

JavaScript makes websites interactive by adding functions and dynamic content.

API

APIs allow different programs and websites to exchange information.
